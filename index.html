How do I join?

Game version: 1.20.6, To change version, change the URL to be java1-20-5.dorite.eu (or whichever version you need. Versions 1.20.1 - 1.20.6 are supported)

To join one of our Java Edition servers, click on the Multiplayer tab in the Main Menu, click either Add Server or Direct Connection, and in the "IP/Web Address" Input, enter java.dorite.eu

What about Bedrock?
We have plans to release some Bedrock servers. When we do, you will be able to access them by pressing "Play", "Servers", scroll down and click "Add Server", then enter bdrk.dorite.eu
